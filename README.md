# Zoo

An OCaml implementation of the Monkey programming language from the book
[Writing An Interpreter in Go](https://interpreterbook.com/). I chose the name
"Zoo" as it came after the word "Monkey" in [this
video](https://www.youtube.com/watch?v=u5BfaguVCBw).

Currently, I intend on following along with the book to implement all features
of the language, before adding advanced features such as compiling to bytecode
and JIT.
